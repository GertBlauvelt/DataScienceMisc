#Jabba is a really annoying outlier that makes it hard to see anything else on the graph with much detail, so this is a little filter to drastically improve this dataset in every single way

starwarsnojabba <- starwars%>%
  filter(name != "Jabba Desilijic Tiure")

#I wanted to see what a jitter plot does, turns out its just a point plot, but with a little bit of randomness added, so that for very dense data, the points are slightly more spread out

ggplot(data = starwarsnojabba, mapping = aes(x = mass, y = height, color = sex)) + 
  geom_jitter()
Warning: Removed 28 rows containing missing values (`geom_point()`).

from this graph, we can see that:

  1. there are way more males than females in this data set

  2. There is a fairly linear connection between height and mass, as one increase, so does the other, unsurprisingly.

  3. Males have a broader range, with the cluster of the shortest and lightest characters being almost exclusively male. The tallest and heaviest are also males.

  4. Withing the main cluster of characters, we can see that males are pretty much all heavier than the female characters, but that they aren’t that much taller.
